# 太阳系模拟系统

一个基于 Three.js 的交互式 3D 太阳系模拟系统，包含太阳及八大行星的精确模拟。

## 功能特性

### 核心功能
- ✅ **精确的天体系统**：包含太阳及八大行星，尺寸和距离按比例缩放
- ✅ **行星运动**：行星具备公转（绕太阳）和自转（绕自身轴）运动，速度设置合理
- ✅ **特殊效果**：土星环、地球月球系统
- ✅ **视觉效果**：
  - 逼真的纹理贴图（程序化生成）
  - 动态光照（太阳作为主要光源）
  - 轨道可视化
  - 星空背景
- ✅ **交互控制**：
  - 鼠标拖拽旋转视角
  - 滚轮缩放
  - 点击行星查看信息
  - 快速聚焦特定行星
- ✅ **动画与时间控制**：
  - 平滑的动画效果
  - 可调整模拟速度（0-10倍速）
  - 暂停/继续功能
  - 重置功能
- ✅ **信息显示**：点击行星显示基本信息（名称、直径、质量等）

### 扩展功能
- ✅ 精确的天文数据（轨道距离、行星尺寸、公转周期等）
- ✅ 响应式设计，支持移动设备
- ✅ 可切换显示轨道线和标签

## 使用方法

### 方法1：使用启动脚本（推荐）

**Windows用户：**
- 双击运行 `start.bat` 或 `start.ps1`
- 选择启动方式（Python/Node.js/直接打开）

### 方法2：使用本地服务器

**使用 Python：**
```bash
python -m http.server 8080
# 或
py -m http.server 8080
```
然后在浏览器中访问 `http://localhost:8080`

**使用 Node.js：**
```bash
npx http-server -p 8080
```
然后在浏览器中访问 `http://localhost:8080`

### 方法3：直接打开（可能有限制）

直接双击 `index.html` 文件在浏览器中打开。注意：某些浏览器可能因为安全策略限制本地文件访问。

### 端口说明

如果8080端口被占用，可以尝试其他端口：
```bash
python -m http.server 3000  # 使用3000端口
```

## 操作说明

### 鼠标控制
- **左键拖拽**：旋转视角
- **滚轮**：缩放场景
- **右键拖拽**：平移视角
- **点击行星**：查看行星信息并自动聚焦

### 控制面板
- **时间速度滑块**：调整模拟速度（0-10倍速）
- **暂停按钮**：暂停/继续动画
- **重置按钮**：重置所有行星位置
- **显示轨道**：切换轨道线显示
- **显示标签**：切换行星名称标签

### 快速聚焦
使用底部的行星选择器快速跳转到特定行星。

## 技术栈

- **Three.js r128**：3D 图形渲染
- **原生 JavaScript**：无需构建工具
- **HTML5/CSS3**：现代化界面

## 天文数据

系统使用相对比例的天文数据，包括：
- 行星尺寸（相对太阳）
- 轨道距离（相对比例）
- 公转速度（相对速度）
- 自转速度
- 行星颜色（基于真实外观）
